🩺 AI Doctor Fine-Tuning Project

A project focused on fine-tuning a Large Language Model (LLM) to act as an AI-powered doctor assistant capable of answering medical-related queries. The project demonstrates modern parameter-efficient fine-tuning techniques using LoRA, Unsloth, Hugging Face Transformers, and TRL.

✨ Features

Medical instruction tuning

Custom dataset preparation

Parameter-efficient fine-tuning with LoRA

GPU-optimized training with Unsloth

Model inference for medical Q&A

Organized project structure for experimentation

🛠️ Tech Stack

Python

Core programming language

Unsloth

Fast fine-tuning and memory optimization

Transformers

Model loading and training utilities

TRL

SFTTrainer for supervised fine-tuning

PEFT (LoRA)

Parameter-efficient adaptation

PyTorch

Deep learning backend
